Instructions

This is a custom VISION26 Pattern Generator, a generative design tool developed for the Motive Vision identity. It creates animated, interactive pattern compositions in real time, which can be exported as digital assets for use across project applications.

Patterns

Choose from four pattern types, each representing a part of Motive:

  • Pathways — Flowing lines that reflect Motive's logistics moving across roads.
  • Terrain — Layered lines inspired by agriculture and natural landscapes.
  • City — A grid of streets and blocks representing urban logistics.
  • Networks — Connected points showing technology and innovation.

Themes

Switch between Light, Dark, and two Color modes. Use Invert to flip the colors.

Motion

  • Animate Lines — Turn line movement on or off.
  • Flat — Pattern stays in place.
  • Move 1 — Slow horizontal drift, looping smoothly left and right.
  • Move 2 — Figure-eight drift, moving in both horizontal and vertical axes.
  • World — The pattern wraps around a rotating globe.

Controls

  • Speed — Adjust how fast the pattern moves.
  • Density — Control how full or minimal the pattern feels.
  • Seed — Generate a new version of the pattern.
  • Reset — Return everything to default.

Export

  • PNG / JPG — Save a still image.
  • SVG — Export a scalable file.
  • Animation — Record a looping video. Duration matches the natural cycle of each spatial mode.
  • Intro — Lines draw in from empty, then play the full animation loop.
  • Outro — Play the full animation loop, then lines draw out to empty.
